# Mayondo Wood & Furniture

A full-stack web application where workers get to keep track and record wood and furniture products. Built with Node.js, Express, MongoDB, and Pug templating.

## Features

- Browse a catalog of wood and furniture products
- View detailed product pages
- Shopping cart and order management
- Server-side rendered pages with Pug templates
- RESTful API routes with Express middleware
- MongoDB database for products and orders

## Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Backend    | Node.js, Express.js |
| Database   | MongoDB             |
| Templating | Pug                 |
| Frontend   | HTML, CSS, JavaScript |

## Project Structure

```
mayondoWoodAndFurniture/
├── html/           # Static HTML assets
├── middleware/     # Express middleware
├── models/         # MongoDB models
├── public/         # Static files (CSS, JS, images)
├── routes/         # Express route handlers
├── views/          # Pug templates
├── .gitignore
└── server.js       # Entry point
```

## Getting Started

### Prerequisites
- Node.js v14+
- MongoDB

### Installation

```bash
git clone https://github.com/treasure2005-bit/mayondoWoodAndFurniture.git
cd mayondoWoodAndFurniture
npm install
```

### Running the app

```bash
# Development
npm run dev

# Production
npm start
```

The app will run at `http://localhost:3000`
